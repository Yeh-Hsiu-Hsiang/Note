# 標題
### Hello  
world

```語法: # H1 ```

##### 最多只有六個 **#**
##### 文字後須空兩格才能換行

```
語法:
    粗體:  **txt** or __txt__
    斜體:   *txt* or _txt_
    刪除線:  ~~txt~~    
```

# 區塊
#### 小區塊
``` print('Hi') ```

```
語法: ```txt```
```


#### 大區塊

``` python
print('Hello, world')
```

```語法: 把小區塊的語法拆成三行```


#### 程式碼： **在```後(不換行) 可加上欲使用之語言名稱**


```
例:
    ``` python
    print('Hello')
    ```
```

# 表格

| title | title | title |
| ----- | ----: | ----: |
|  內容  | abcd |  dfgh |

```
語法:

|  title |  title |  title |
|  對齊  | -----: |  --:-- |
|  內容  |  內容  |  內容   |         

```
#### **:** 表示對齊方向(左邊靠左)
#### 外圍 **|** 可省略，必需是半形的。
