### ssh  
```  
ssh 帳號@主機 IP  

-p 指定連線 port 號
```  
ex : super@10.20.30.5    

### scp  

```  
scp [Options] file name@ip:目錄       --> 上傳  
scp [Options] name@ip:file 本機目錄   --> 下載  

  -p 保留檔案權限  
  -r 複製整個目錄  
```

查詢紀錄  
.zsh_history
